## 🛠️ Troubleshooting

If you run into issues, here are quick fixes you can try:

- **Cannot link Telegram account:** Verify you are using the correct Telegram app and have allowed the required permissions.
- **Installation fails:** Restart your computer and try running the installer as an administrator.
- **Bot disconnects:** Re-link your Telegram account by restarting the app and scanning the QR code again.
- **Commands return errors:** Confirm you send clear, simple instructions as described in the usage section.
